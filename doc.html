4. activity-list

作者：

请求说明

请求方式：POST
请求URL ：/activity/list

请求参数

字段 字段类型 必填 字段说明
details slice[struct] 是 活动详情列表 req-details
page int 是 页码：从1开始
status int 是 活动状态: -1:表示查全部;1-准备;2-关闭,oneof=-1 1 2

req-details
字段 字段类型 必填 字段说明
aid uint64 是 活动id
titleName string 是 活动标题
startTime string 是 活动开始时间
endTime string 是 活动结束时间
status uint8 是 活动状态

返回结果

 {
	"code": 0,
	"data": {
		"totalCount": 0,
		"details": [
			{
				"aid": 0,
				"titleName": "",
				"startTime": "",
				"endTime": "",
				"status": 0
			}
		],
		"goods": [
			{
				"gid": 0,
				"goodsName": ""
			}
		]
	},
	"msg": ""
}

返回参数

字段 字段类型 字段说明
code int 0-成功 1-失败
data interface rsp-data
msg string 错误信息

rsp-data
字段 字段类型 字段说明
totalCount int64 总数
details slice[struct] 活动详情列表 rsp-details
goods slice[struct] 商品详情列表 rsp-goods

rsp-details
字段 字段类型 字段说明
aid uint64 活动id
titleName string 活动标题
startTime string 活动开始时间
endTime string 活动结束时间
status uint8 活动状态

rsp-goods
字段 字段类型 字段说明
gid uint64 商品id
goodsName string 商品名称
